# Chaldur
Changes the Balatro challenge menu to be easier to use, similar to the Galdur mod.

## HEADS UP!
This mod takes a lot of inspiration, and a lot of code, from Eremel's Galdur mod. I am teaching myself Balatro mod development and Lua through this project, which I'm proud of, but it would not be possible without the work and code of those who came before.
